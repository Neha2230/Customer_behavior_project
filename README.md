End-to-End Data Analysis Project
📌 Overview

This project demonstrates a complete end-to-end data analysis workflow, starting from raw data loading in Python to building an interactive Power BI dashboard and presenting insights through a structured report and presentation.

The objective was to transform raw data into actionable insights using industry-standard tools and analytical best practices.

📂 Dataset

Structured dataset (CSV format)

Includes categorical and numerical variables

Contains missing values and inconsistencies requiring cleaning

Used for business-focused analytical insights

(Dataset included in the /data folder of this repository.)

🛠️ Tools & Technologies

Python – Data loading, cleaning, and EDA

Pandas & NumPy – Data manipulation

Matplotlib & Seaborn – Data visualization

PostgreSQL – SQL querying and data analysis

Power BI – Interactive dashboard creation

Gamma – Presentation development

Jupyter Notebook – Analysis environment

🔎 Project Steps
1️⃣ Data Loading (Python)

Imported dataset using Pandas

Explored dataset structure (shape, data types, summary statistics)

2️⃣ Data Cleaning

Handled missing values

Removed duplicates

Standardized column formats

Treated inconsistencies and outliers

3️⃣ Exploratory Data Analysis (EDA)

Univariate and bivariate analysis

Identified trends and patterns

Correlation analysis

Created visualizations to support findings

4️⃣ SQL Analysis (PostgreSQL)

Imported cleaned dataset into PostgreSQL

Wrote SQL queries for:

Aggregations

Grouping and filtering

Business KPI calculations

Trend analysis

5️⃣ Dashboard Development (Power BI)

Connected Power BI to PostgreSQL

Built interactive dashboard with:

KPI cards

Trend charts

Category breakdowns

Filters & slicers for dynamic insights

6️⃣ Reporting & Presentation

Created a structured analytical report summarizing findings

Designed a presentation using Gamma to communicate insights clearly

📊 Dashboard

The Power BI dashboard includes:

Key performance indicators (KPIs)

Trend analysis visuals

Category and segment comparisons

Interactive filtering options

(Dashboard file available in /dashboard folder.)

📈 Results

Cleaned and structured raw data for analysis

Identified key business trends and performance drivers

Built an interactive dashboard for stakeholder decision-making

Delivered insights through a professional report and presentation

This project demonstrates practical skills in data cleaning, SQL querying, business analysis, and data visualization.

🚀 How to Run
1️⃣ Clone the Repository
git clone <repository-link>
2️⃣ Install Required Python Libraries
pip install pandas numpy matplotlib seaborn psycopg2
3️⃣ Run Jupyter Notebook
jupyter notebook

Open and execute the analysis notebook.

4️⃣ PostgreSQL Setup

Import cleaned dataset into PostgreSQL

Run SQL scripts provided in /sql folder

5️⃣ Open Power BI

Open .pbix file from /dashboard

Refresh data connection
